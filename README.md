# Internshala-automation-tool
A browser automation tool built with Node.js and Puppeteer to streamline the internship application workflow on Internshala by automating profile completion and application submission.

**Overview**

This project demonstrates practical use of browser automation for repetitive web workflows, including authentication, form filling, navigation, and dynamic DOM interaction.

It was developed to reduce manual effort in applying to multiple internships while showcasing automation engineering and web scraping/browser scripting skills.

**Key Features**
Automated login to Internshala
Profile completion/update automation
Graduation details
Training/Experience details
Portfolio/Work Sample links
Internship listing navigation
Dynamic application form handling
Automated response filling for screening questions
Batch internship application submission
Technical Highlights
Browser automation using Puppeteer
Asynchronous workflow management with async/await
Dynamic DOM scraping and selector handling
Multi-step form automation
Config-driven data injection via external JSON/JS files
Real-world automation of production website workflows
